## Development

### Setup

- setup a PostgreSQL server
- copy `.env.example` to `.env` and fill in all variables
- install dependencies: `yarn`
- start webpack: `yarn dev`

### Debugging (VS Code)

- run `yarn watch` in a terminal
- run `dev` configuration
